# BabyEnes

Baby cry analysis and deep learning experiments.

## Included in this repository

- Python scripts and dependency file
- Jupyter notebooks used for training/evaluation
- Research/support CSV files in `content/`
- Analysis scripts in `scripts/`

## Excluded from version control

To keep the repository portable and within common Git host limits, large/local artifacts are ignored:

- Local virtual environment directories
- Trained model binaries in `models/`
- Raw/derived datasets in `data/`
- Zip archives and large binary files
- Apple Numbers files

## Quick start

1. Create and activate a Python environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run scripts or open notebooks as needed.

## Notes

If you want to version large assets (for example trained models), use Git LFS.
